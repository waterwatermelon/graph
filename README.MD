## 文件结构说明
- IOT         物联网管理平台
- CDRS        话单计费系统
- BAS         业务接入系统
- AI-Meeting  智慧会议系统
- CSS         体检监管系统
- MMS         模具管理系统
- ODS         在线文档系统
- PTS         产品定制任务系统

- SDN         描述SDN、openflow相关设计
- Jtopo.drawio 描述jtopo.js库文件的设计
- gw-mtk      网关设计
